# Electronics Projects

This repository contains hands-on electronics experiments performed while studying fundamental electrical engineering concepts.

The goal of these projects is to connect theoretical circuit analysis with practical circuit implementation and measurement.

---

## Projects

### 1. Voltage Divider Experiment

This project demonstrates the voltage divider rule and investigates the effect of loading on the output voltage.

Concepts explored include:

* Ohm's Law
* Kirchhoff’s Voltage Law
* Voltage Divider Rule
* Loading Effect in resistive networks

Project files can be found in the folder:
