# Voltage Divider Experiment

## Objective

To experimentally verify the voltage divider rule and observe the effect of loading on the output voltage.

---

## Components Used

* Breadboard
* Resistors
* Multimeter
* 9V power supply

---

## Circuit Description

Two resistors were connected in series to form a voltage divider. The output voltage was measured at the node between the resistors.

Additional load resistors were connected between the output node and ground to observe how the output voltage changes due to loading.

---

## Experiments Performed

1. Measured actual resistor values using a multimeter.
2. Calculated theoretical output voltage using the voltage divider formula.
3. Measured voltage across each resistor.
4. Tested the effect of different load resistors on the output voltage.

---

* Applying Ohm's Law in practical circuits
* Verifying Kirchhoff’s Voltage Law experimentally
* Breadboard circuit prototyping
* Using a multimeter to measure voltage, current, and resistance
* Understanding the loading effect in voltage divider circuits
* Documenting engineering experiments
